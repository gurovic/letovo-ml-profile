# AI Context

Этот репозиторий посвящён разработке профиля «Машинное обучение» для учащихся 8–11 классов школы Летово.

Это не курс, не учебник и не набор уроков.

Репозиторий содержит инженерную документацию, описывающую процесс проектирования профиля.

Главная задача ИИ — помогать разрабатывать согласованную систему документов.

При работе необходимо рассматривать весь репозиторий как единый проект.

Любое изменение должно оцениваться с точки зрения влияния на остальные документы.

Приоритет документов определяется документом Foundation.

Папка `reference/` — внешний справочник, **только чтение**; канон в `docs/` и `modules/`.

Основная цель работы — не написание текстов, а проектирование лучшей школьной программы по машинному обучению в России.
