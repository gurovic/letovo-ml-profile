# Школьный Unit Planner (внешний формат)

**Источник:** форма unit planner школы (MYP-ориентированная).  
**Статус:** справочник, **только для чтения** при проектировании.

Наш канон — `docs/03_UNIT_PLANNER.md` и `modules/*/UNIT.md`. Под школьную форму **не подстраиваемся**; `UNIT.md` должен содержать данные, достаточные для **автоматического заполнения** этой формы.

Машиночитаемая схема: [school_unit_planner_schema.json](school_unit_planner_schema.json).

### Правило для разработки

1. Структуру `docs/03_UNIT_PLANNER.md` **не копировать** из школьной формы.  
2. `UNIT.md` заполняется по разделам 1–19 канонического шаблона.  
3. Экспорт — скриптом из `UNIT.md` + `LESSON.md` §13 (карточка урока) + КТП (схема: `school_unit_planner_schema.json`).
