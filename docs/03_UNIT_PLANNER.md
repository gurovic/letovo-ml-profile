## Назначение

Unit Planner отвечает на один вопрос:

**Как проектируется модуль?**

Документ заполняется **до** создания уроков, ноутбуков, презентаций и активностей.

Общие правила проектирования материалов — в Material Design Standard. Здесь — **шаблон модуля**.

---

## Как пользоваться

1. Пройти обязательные вопросы из Material Design Standard (раздел 2).
2. Заполнить поля ниже.
3. Проверить по чек-листу модуля (раздел 12).
4. Проверить по общему чек-листу Material Design Standard (раздел 11).

### Длительность

| Компонент | Правило |
|---|---|
| Уроки содержания | столько, сколько нужно для результатов обучения (типично 6–8 при 2 ч/урок) |
| Итоговая работа | **до 2 уроков**: проектирование/черновик + доработка/сдача |
| Итого | при необходимости **до 10 уроков** на модуль |

Итоговая работа — не «ещё одна лекция», а время на артефакт из раздела 10.

## 13. Чек-лист модуля

- [ ] модуль имеет ясную цель;
- [ ] каждый урок вносит вклад в цель;
- [ ] ни один урок не существует «по традиции»;
- [ ] пререквизиты достаточны;
- [ ] результаты обучения соответствуют активностям;
- [ ] инженерная практика присутствует;
- [ ] математика обоснована;
- [ ] итоговый артефакт демонстрирует заявленные результаты;
- [ ] модуль согласован с траекторией класса (Foundation, раздел 7).
