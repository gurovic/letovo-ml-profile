# KTP (календарно-тематическое планирование)

Статус: Draft 1

## Назначение

Отвечает на вопрос: **как спланирован учебный год профиля по парам?**

КТП — операционный документ. Живёт в **`docs/ktp/`**, не в `reference/`.

| Правило | Смысл |
|---|---|
| 1 тема = 1 пара | 2 академических часа (учёт) = **80 минут** проведения; **тема** — либо введение, либо отработка, либо интеграция/артефакт |
| Нагрузка 8–9 | 5 ч/нед |
| Нагрузка 9–10 | 6 ч/нед |
| Отработка в КТП | для программных навыков — явные строки «Практика: …» или эквивалентный блок; новая конструкция без отработки — ошибка плана (Foundation §5, Pedagogy §2) |
| Итоговая работа модуля | до 2 пар (см. Unit Planner); не заменяет первичную отработку |
| Сюжет модуля | свой контекст; не «продолжение» предыдущего модуля |
| Вводная пара | входит в **модуль 1** (пара 1 КТП): ИИ, ML, профиль, организация обучения |

Подробности нагрузки — Foundation §7.

## Связь с модулями

Строка КТП → `modules/<grade>_<nn>_<slug>/` (Unit Planner).

**8 класс:** 11 модулей, 85 пар — индекс в [README.md](../README.md#модули-8-класса); КТП: [ktp/08.md](ktp/08.md).

Пример: пары 1–10 → [modules/08_01_functions_recursion/](../modules/08_01_functions_recursion/) (вводная + функции); пары 11–20 → [modules/08_02_carsharing_pandas_lr/](../modules/08_02_carsharing_pandas_lr/).

### Матрица навыков

| Правило | Смысл |
|---|---|
| Файл | [ktp/08_skills_matrix.md](ktp/08_skills_matrix.md) — строки пар класса, колонки «Узнали / Отработали» по EDA, ML, Python, алг. |
| Когда обновлять | Вместе с модулем: новый или изменённый `UNIT.md`, `LESSON.md`, ноутбуки, артефакт — если меняются вводимые или отрабатываемые навыки |

## Карта инструментов

**Вопрос:** с какой пары (модуля) можно использовать конструкцию Python, приём анализа данных, идею ML или математику в материалах — без забегания вперёд?

| Правило | Смысл |
|---|---|
| **Первая введение** | С этой пары КТП инструмент появляется в учебных материалах; на более ранних парах — **нельзя** (исключение: устная ориентация на вводной) |
| **С пары N** | Можно использовать в ноутбуке и заданиях без отдельного введения |
| **7 класс** | Пререквизит основного курса (циклы, `if`, списки, базовый Python) |
| **Specмат** | Параллельный курс спецматематики; в профиле — прикладно, без строгой теории, если не сказано иначе |

Детальные таблицы по **8 классу** (по парам и модулям): [ktp/08.md — Карта инструментов](ktp/08.md#карта-инструментов).

При проектировании ноутбука и `UNIT.md` §14 сверяться с картой. Общие правила формата — Notebook Standard §3.
