## Правила заполнения

| Правило | Смысл |
|---|---|
| Узнали | новая идея на паре введения или интеграции |
| Отработали | серия задач на уже введённый навык; на паре «Практика: …» — в основном эта колонка |
| Без дубля | один термин — либо «Узнали», либо «Отработали» в той же строке, не в обеих |
| Термины | общеупотребительные названия идей; без имён функций, сюжетных данных и внутренних меток урока |
| EDA | описание и преобразование данных без fit модели |
| ML | predict, метрики, pipeline, классификация |
| Python | синтаксис и инженерия кода |
| Алг. и стр. данных | алгоритмы, сложность, структуры, рекурсия, DP |

## Обновление при изменении модуля

| Когда | Что сделать |
|---|---|
| Новый модуль или переработка `UNIT.md` §11 | Заполнить строки матрицы для всех пар модуля |
| Изменён `LESSON.md`, ноутбук, артефакт | Проверить строку соответствующей пары; обновить колонки, если сдвинулись «Узнали» / «Отработали» |
| Переименована тема в [08.md](08.md) | Синхронизировать столбцы «Модуль» и «Урок» в матрице |
