Карта годов

Абстракции собраны в семейства (курируемая таксономия). Карта годов показывает, в каком классе какое семейство пополняется. Клик по концепции — её жизнь по годам.

Как смотреть
  1. Сверху — система семейств (M / P / L). Клик по семейству подсвечивает его рождения по годам.
  2. Пять колонок = классы. Внутри — не случайный список, а блоки семейств с кратким смыслом.
  3. Внутри семейства концепции идут от более простых к сложным (слой пререквизитов).
  4. Клик по коду → путь «рождение / возвращается» и модули нашего 8 класса.
M математика P программирование L машинное обучение Таксономия: concept_taxonomy.json

Данные: python scripts/build_concept_rivers_data.py (reference/program_data.json + concept_taxonomy.json). Другой вид: реки семейств. Канон — docs/ и modules/.
