# Lesson Design

Статус: Draft 3

## Назначение

Lesson Design отвечает на один вопрос:

**Как проектируется урок?**

Документ заполняется **до** создания ноутбуков, презентаций, активностей и домашних заданий.

Общие правила — в Material Design Standard. Контекст модуля — в Unit Planner. Здесь — **шаблон урока**.

## 8. Активности

| # | Активность | Вид активности | Зачем | Артефакт | Формат работы | Сложность |
|---|---|---|---|---|---|---|
| 1 | | | | | индивидуально / пара / группа | |

**Вид активности** — что делает учащийся (не формат материала). Типичные значения:

- написание кода
- отладка
- эксперимент с данными
- анализ кода или вывода
- обсуждение / аргументация
- проектирование (схема, API, критерии)
- проверка / тестирование

Одна строка — одна завершённая активность с артефактом.
