# Lesson Design

Статус: Draft 2

## Назначение

Lesson Design отвечает на один вопрос:

**Как проектируется урок?**

Документ заполняется **до** создания ноутбуков, презентаций, активностей и домашних заданий.

Общие правила — в Material Design Standard. Контекст модуля — в Unit Planner. Здесь — **шаблон урока**.

---

## Как пользоваться

1. Пройти обязательные вопросы из Material Design Standard (раздел 2).
2. Заполнить поля ниже.
3. Проверить по чек-листу урока (раздел 12).
4. Проверить по общему чек-листу Material Design Standard (раздел 11).
