## 2. Зачем существует модуль

**Проблема:** kNN (модуль 4) не даёт интерпретируемую вероятность и плохо масштабируется; без venv и README эксперименты невоспроизводимы.

**Если убрать модуль:** нет моста к вероятностным моделям 9 класса; инженерная дисциплина среды не формируется.

## 5. Результаты обучения

После модуля учащийся может:

1. **Объяснить** логистическую регрессию как модель вероятности класса через сигмоиду.
2. **Обучить** `LogisticRegression`: fit/predict, выбор порога.
3. **Вычислить** метрики на несбалансированных классах.
4. **Запустить** скрипт из CLI, навигировать в терминале.
5. **Создать** venv и `requirements.txt`.
6. **Оформить** README эксперимента для воспроизводимости.
