# Unit Planner: логистическая регрессия и venv

**Класс:** 8  
**Статус:** план модуля; `LESSON.md` — после финализации группировки пар  
**Длительность:** 5 пар по КТП (10 академических часов)

**Сквозная линия модуля:** сигмоида → logreg fit/predict → метрики дисбаланса → CLI → venv → README эксперимента.

**Данные (канон для будущих LESSON):** [UCI Bank Marketing](https://archive.ics.uci.edu/dataset/222/bank+marketing) — отклик на телемаркетинг (`y`). Столбец **`duration` запрещён** как признак (leakage). Сюжет телекома «СвязьОнлайн» не использовать как основной датасет.

## 2. Зачем существует модуль

**Проблема:** kNN (модуль 4) не даёт интерпретируемую вероятность и плохо масштабируется; без venv и README эксперименты невоспроизводимы.

**Если убрать модуль:** нет моста к вероятностным моделям 9 класса; инженерная дисциплина среды не формируется.

## 5. Результаты обучения

После модуля учащийся может:

1. **Объяснить** логистическую регрессию как модель вероятности класса через сигмоиду.

5. **Создать** venv и `requirements.txt`.
6. **Оформить** README эксперимента для воспроизводимости.

### Бриф для design-lesson

| Правило | Содержание |
|---|---|
| Бизнес-вопрос | Кому звонить в первую очередь при ограниченном бюджете обзвона? |
| Leakage | `duration` — длительность уже состоявшегося звонка; **нельзя** в X для продакшен-логики и для сдачи |
| Линии | **describe** (доля yes) → **transform** (простые кодировки категорий) → **predict** (logreg, порог, F1/recall) |
| Дисбаланс | Не оценивать только accuracy |
| Этикетка «churn» | В материалах говорить «отклик / подписка на депозит»; слово churn в названии папки модуля историческое — в текстах уроков не путать с уходом из телекома |
| Мост к M7 | Банк «Надёжный» (поиск в логах) — только нарративный мост, навыки не из M7 |

## 18. Глоссарий модуля

| Термин | Определение в контексте модуля |
|---|---|
| Отклик (`y`) | Клиент согласился на срочный депозит в кампании (yes/no) |
| Leakage | Признак, известный только после события, которое предсказываем (здесь — `duration`) |
| Сигмоида | S-образная функция, сжимающая значение в (0, 1) |
| `predict_proba` | Вектор вероятностей классов |
| Порог (threshold) | Граница отнесения к положительному классу |
| venv | Изолированное окружение Python с своими пакетами |
| requirements.txt | Список зависимостей для воспроизведения |
| Воспроизводимость | Повторение результата по документации |
