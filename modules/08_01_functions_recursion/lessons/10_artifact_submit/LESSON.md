## D. Проектирование

### Зачем урок

Закрывает модуль **сдачей**: сравнение классов текстов, наивная классификация, README и рефлексия — проверяемый итог 10 пар.

### Центральная идея

| Поле | Значение |
|---|---|
| Центральная идея | Готовый артефакт = 10 зелёных тестов + README с решениями ученика |
| Данные | `TEXTS_POSITIVE`, `TEXTS_NEGATIVE` |

### Результаты обучения

1. Реализовать шаги 6–8 из starter/README.
2. Пройти все 10 тестов `manual_tests.py`.
3. Написать README: 5 фраз для `naive_classify` (≥4 верных), pipeline, tie-break.
