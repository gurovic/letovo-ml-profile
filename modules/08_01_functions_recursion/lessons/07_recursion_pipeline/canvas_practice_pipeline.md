# Практика из Canvas (ML-адаптация)

Задачи из контеста «Лямбда-головоломка» (IT-8, Canvas), переформулированы под data/ML.

---

## A. Наиболее «удалённый» объект

`FEATURE_POINTS` — пары `[area_sqm, floor]`. Найти точку с **максимальной** суммой квадратов координат (без sqrt).

Подсказка: `max(FEATURE_POINTS, key=lambda p: ...)`

---

## B. Сортировка id по последней цифре метрики

После `int(f1 * 100)` отсортировать id прогонов по **последней цифре** (stable sort).

---

## C. Leaderboard моделей

`MODEL_RUNS`: sort по **убыванию f1**; при равенстве — по **возрастанию id**.

---

## D. Среднее трёх метрик

Список `{name, m1, m2, m3}` — sort по убыванию `(m1+m2+m3)/3`.

---

## E. filter + map

`EXAM_SCORES`: значения ≥ 60 → доля от 100.

---

## Не включено (IT-only)

Сортировка по сумме цифр, чётные/нечётные, задачи G/H.
