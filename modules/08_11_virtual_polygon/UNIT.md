# Unit Planner: производная и градиентный спуск

**Класс:** 8  
**Статус:** план модуля; `LESSON.md` — после финализации группировки пар  
**Длительность:** 4 пар по КТП (8 академических часов)

**Сквозная линия модуля:** производная как скорость изменения → численный градиентный спуск → интеграл обзорно → связь с минимизацией loss.

**Обязательный модуль:** интуиция производной и интеграла для 9 класса (КТП, Foundation §7).

## 2. Зачем существует модуль

**Проблема:** `model.fit()` в sklearn остаётся чёрным ящиком без идеи «движения к минимуму ошибки»; без производной нет связи с курсом спецматематики.

**Если убрать модуль:** разрыв между регрессией (модули 2, 6) и формальным анализом в 9 классе; слабая мотивация градиентных методов.

## 5. Результаты обучения

После модуля учащийся может:

1. **Интерпретировать** производную как скорость изменения функции (на примерах и графике).
2. **Реализовать** численный градиентный спуск для минимизации 1D-функции.
3. **Объяснить** интеграл как площадь под кривой (обзорно, без техники интегрирования).
4. **Связать** шаги градиентного спуска с обучением модели (движение к минимуму loss).

## 9. Математические идеи

| Идея | В модуле |
|---|---|
| Производная | предел отношения приращений (интуитивно) |
| Численная производная | (f(x+h) − f(x)) / h |
| Градиентный спуск | x ← x − η·f′(x) |
| Интеграл | площадь под графиком (обзорно) |
| MSE как loss | связь с регрессией |

## 18. Глоссарий модуля

| Термин | Определение в контексте модуля |
|---|---|
| Производная | Мгновенная скорость изменения функции; наклон касательной |
| Градиентный спуск | Итерационное движение к минимуму по направлению убывания |

| Численная производная | Приближение производной через малый шаг h |
