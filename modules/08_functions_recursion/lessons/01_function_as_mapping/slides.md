## Слайд 3 — Ключевая идея

**Математика:** price = f(area) = 1.5 + 0.09 × area

**Python:**

```python
def predict_price(area_sqm):
    return 1.5 + 0.09 * area_sqm
```

**sklearn (позже):** `model.predict([[area]])` — тот же смысл.
