# Данные модуля 2

| Файл | Назначение |
|---|---|
| `trips.csv` | Сквозная таблица поездок каршеринга «Дорога» (200 строк) |
| `generate_trips.py` | Пересборка CSV с фиксированным seed (42) |

## Столбцы

| Столбец | Роль |
|---|---|
| `trip_id` | Идентификатор (не признак) |
| `distance_km` | Признак |
| `duration_min` | **Цель** — длительность поездки, мин |
| `hour` | Признак — час начала |
| `zone` | Категориальный признак (фильтры на паре 13) |
| `vehicle_type` | Категориальный признак |

Пересборка:

```bash
python modules/08_02_carsharing_pandas_lr/data/generate_trips.py
```
